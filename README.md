# diskgover

> **Note:** This is a learning project to improve Go skills. It may or may not evolve into a full rewrite of [diskover](https://github.com/diskoverdata/diskover-community). Expect experiments, iterations, and breaking changes.

## About

diskgover is a personal exploration into building disk indexing and analysis tools with Go. The project serves as a hands-on way to learn:

- Go project structure and best practices
- File system operations and traversal
- Concurrent programming patterns
- Performance optimization
- Data indexing and storage strategies
